# 优客商城概要设计

## 项目结构

```
youke-shop 优客商城父项目
 --youke-dao 公共dao层
 --youke-pojo 公共实体层
 --youke-service 公共service层
 --youke-common 公共工具层
 --youke-api api接口层
 --youke-web web界面层
 --youke-admin 后台管理系统
```

## 技术选型

前台：layui
后台：springboot+mybatis plus+mysql+shiro
